# Zadatak 1: Sabiranje brojeva u Pythonu

Dobrodošli na prvi zadatak! Cilj ove vežbe je upoznavanje sa osnovnom sintaksom Python programskog jezika, kao i procesom preuzimanja i predaje koda putem GitHub Classroom-a.

## Opis zadatka
U repozitorijumu se nalazi fajl `main.py`. Unutar njega je definisana funkcija `saberi_brojeve(a, b)` koja trenutno ne radi ništa (sadrži samo ključnu reč `pass`). 

Vaš zadatak je da implementirate logiku unutar ove funkcije tako da ona vraća matematički zbir prosleđenih argumenata `a` i `b`.

## Koraci za izradu
1. Klonirajte ovaj repozitorijum na svoj računar ili ga otvorite u editoru koda (npr. Visual Studio, IntelliJ IDEA ili VS Code).
2. Otvorite fajl `main.py`.
3. Pronađite funkciju `saberi_brojeve(a, b)`.
4. Obrišite `pass` i napišite odgovarajući kod koji vraća rezultat.
5. Sačuvajte izmene u fajlu.

## Kako da testirate kod?
Pre nego što predate zadatak, proverite da li vaš kod radi. Otvorite terminal, pozicionirajte se u folder gde se nalazi `main.py` i pokrenite sledeću komandu:

`python main.py`

Ako ste zadatak uspešno rešili, skripta će ispisati tačan zbir brojeva koji su navedeni u test bloku na dnu fajla.

## Predaja zadatka
Kada se uverite da program radi ispravno, potrebno je da pošaljete svoje izmene nazad na repozitorijum. To radite kroz standardne Git komande (ili grafički interfejs vašeg editora):

1. `git add main.py`
2. `git commit -m "Završen zadatak sabiranja brojeva"`
3. `git push origin main`

Nakon što uradite *push*, osvežite stranicu ovog repozitorijuma u pregledaču kako biste se uverili da je vaš novi kod uspešno postavljen. 

Srećan rad!
